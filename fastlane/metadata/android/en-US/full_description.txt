<p><b>Thor App Manager</b> is a modern, open-source Android app manager and installer, designed for power users and enthusiasts who value control, privacy, and efficiency.</p>

<p>Thor is built 100% in Kotlin, following Material 3 design principles and leveraging Jetpack Compose for a smooth, responsive user experience. With a focus on minimalism, Thor delivers a full-featured app management suite in an APK smaller than 2.20 MB.</p>

<p><b>Key Features:</b></p>
<ul>
  <li>Fully reproducible, copylefted libre software (GPLv3.0)</li>
  <li>Material 3 UI with dynamic color support</li>
  <li>Displays installed apps with advanced sorting and filtering (including by installation source)</li>
  <li>Launch app activities directly</li>
  <li>Install, uninstall, freeze, and unfreeze apps (including system apps)</li>
  <li>Reinstall APKs or apps via Google Play</li>
  <li>Share APK files easily</li>
  <li>Batch operations: reinstall, uninstall, or kill multiple apps at once</li>
  <li>Split APK indicator for apps using multiple APKs</li>
  <li>App state indicator (e.g., frozen = disabled)</li>
  <li>Uninstall and freeze/unfreeze system apps</li>
  <li>Lightweight and fast, with a focus on privacy and no analytics or trackers</li>
</ul>

<p><b>Technical Highlights:</b></p>
<ul>
  <li>100% Kotlin codebase for modern Android development</li>
  <li>Smallest possible APK size for a full-featured app manager</li>
  <li>Uses a modularized, fully Kotlin-converted version of <a href="https://github.com/topjohnwu/libsu">libsu</a> (as <i>suCore</i>) for root operations</li>
</ul>
