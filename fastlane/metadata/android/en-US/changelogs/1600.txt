<h2>Thor v1.6</h2>

<b>What's new:</b>
<ul>
  <li>Major backend improvements for root operations (migrated from <code>libsu</code> to in-app <code>suCore</code> module)</li>
  <li>Initial support for Android 16</li>
  <li>New filters: filter apps by state or installation source</li>
  <li>Sorting options: sort apps by name, install date, last updated, version, etc., in ascending or descending order</li>
  <li>Removed integrity checks, downloadable fonts, and unused dependencies</li>
  <li>Added new grid view for app lists</li>
  <li>Switched to Material Expressive Theme</li>
  <li>Numerous UI tweaks</li>
  <li>Frozen app icons are now greyed out</li>
</ul>

<b>Bug Fixes:</b>
<ul>
  <li>Fixed system app uninstaller bug; system apps can now be uninstalled individually</li>
</ul>

<b>Work In Progress:</b>
<ul>
  <li>Settings UI and more features coming soon</li>
</ul>

<b>APK Size:</b><br>
Release Build -&gt; Less than 2.2MB<br>
Foss Build -&gt; Around 9.1MB (no code and resource compression)
